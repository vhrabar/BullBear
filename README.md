# BullBear
BullBear je brokerska aplikacija za simulirano trgovanje dionicama, opcijama i ETF-ovima uz tržišne podatke u stvarnom vremen.  Projekt je razvijen na FER, UNIZG u sklopu kolegija Programsko inženjerstvo, s ciljem primjene principa timskog rada, dizajna sustava, dokumentiranja, ispitivanja i modernih metoda razvoja.
